# Pokémon Explorer 🚀

Bu proje, PokéAPI kullanarak Pokémonları listeleyen ve arama yapmanıza olanak tanıyan modern bir React uygulamasıdır.

## ✨ Özellikler
- **Geniş Liste:** Tüm Pokémonları görsel kartlar halinde sunar.
- **Canlı Arama:** İstediğiniz Pokémon'u ismine göre anında filtreleyebilirsiniz.
- **Modern Arayüz:** Kullanıcı dostu ve responsive (mobil uyumlu) tasarım.
- **Hızlı Altyapı:** Vite ve React ile optimize edilmiş performans.

## 🛠️ Kullanılan Teknolojiler
- React.js
- Vite
- CSS3 (Custom Styling)
- PokéAPI

## 🚀 Yerel Kurulum

Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/bet403/pokemon.git
   ```
2. Proje klasörüne gidin:
   ```bash
   cd pokemon
   ```
3. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
4. Uygulamayı başlatın:
   ```bash
   npm run dev
   ```
